# WebSelenium

基于selenium 4实现网页数据抓取，本项目仅用于个人测试和学习。<br>

## 使用教程:

1. 安装selenium。

2. 安装Chrome driver。

   安装Chrome driver教程可参考[这里](https://github.com/peilongchencc/selenium_data/tree/main/browser_driver)。<br>

3. 运行examples下的文件，或参考examples下的文件编写适合自己场景的代码。
